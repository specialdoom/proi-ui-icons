proi-ui-icons
---------
icons for proi-ui

## Installation
All controls
```bash
npm i proi-ui-icons
```

## Docs
Check out the [icons](http://proi-ui.com/icons).

## Basic Usage
```javascript
<script>
import { Icon } from 'proi-ui-icons'
</script>

<Icon />
<Icon type="info" />
<Icon type="close" />
```
